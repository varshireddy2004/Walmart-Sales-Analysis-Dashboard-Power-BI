 Project Overview
This project presents an interactive Power BI dashboard built using Walmart sales data to analyze overall sales performance, customer behavior, and product trends.  
The dashboard helps stakeholders make data-driven decisions by providing clear KPIs and visual insights.

Objectives
- Analyze total sales and transaction trends over time  
- Compare sales performance across cities and branches  
- Identify top-performing product lines  
- Understand customer types and payment preferences  
- Evaluate customer satisfaction using ratings  

 Tools & Technologies
- Power BI Desktop
- DAX (Data Analysis Expressions)
- CSV Dataset

---

Dashboard Features
- KPI Cards
  - Total Sales  
  - Total Transactions  
  - Average Sale Value  
  - Average Rating  

- Visual Analysis
  - Sales Trend Over Time (Line Chart)
  - Sales by Product Line (Bar Chart)
  - Sales by City (Column Chart)
  - Sales by Branch (Donut Chart)
  - Payment Method Distribution (Pie Chart)
  - Customer Type Analysis (Bar Chart)
  - Ratings by Product Line (Column Chart)

- Interactive Slicers
  - City  
  - Branch  
  - Customer Type  
  - Payment Method  
  - Date  

Key Insights
- Certain product lines contribute significantly more to total revenue  
- Sales performance varies across branches and cities  
- Member customers generally contribute higher sales  
- Digital payments are widely used  
- Higher-rated product lines show better customer engagement  

DAX Measures Used
- Total Sales  
- Total Transactions  
- Average Sale Value  
- Average Rating  
- Total Gross Income  
- Profit Margin %
